# ScriptPrincess

An updated and working version of the Roblox script for **Princess Tycoon**, originally made by **GGH52Lan**.

## How to Use

Paste the following code into your Roblox executor and run it:

```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/somun2/ScriptPrincess/refs/heads/main/script.lua"))()
```
